# Doom Management System (D.M.S.)

**The new Doom Management System (D.M.S.) is a modular, indestructible command center for retro shooters. This terminal launcher effortlessly manages hundreds of WADs, mods, and engine switches. Equipped with automatic GitHub updates, a dynamic UI, and playtime tracking, it's the ultimate fortress for every Doom Slayer!**

## Features
- 🚀 **Auto-Updater:** Self-updating via GitHub.
- 🎮 **Engine Manager:** Seamlessly switch between GZDoom, UZDoom, Woof, dsda-doom, and more.
- 📂 **Smart Installer:** Automatically detects, moves, and categorizes your WADs.
- 📊 **Stats Tracking:** Keeps track of your playtime and completed maps.

## Installation
1. Download the repository as a ZIP file.
2. Extract it to your preferred gaming folder.
3. Run `start.py` (requires Python installed on your system).
4. Drop your WADs into the `Install` folder and let D.M.S. do the rest!
